# Webcap

精确的浏览器 DOM 级截图工具。

[English](README.md)

## 背景

Webcap 解决的是“截图不够精确”的问题。传统截图只能按屏幕裁剪，而 Webcap 允许你直接选择任意 DOM 元素并输出截图，适用于：

- 设计与研发对比、回归与评审
- 资料整理与知识库沉淀
- 记录真实渲染结果，便于复盘与复用

## 功能

- 鼠标悬停高亮任意 DOM 元素
- 点击或回车捕获当前元素
- 支持当前可视区域截图与全页截图
- 支持“仅复制”或“仅下载”模式
- 可选截图边框、阴影与边距效果
- 内置快捷键与说明面板

## 快捷键

- `Click`：捕获当前元素
- `Enter`：捕获当前元素
- `Esc`：选择父元素
- `Shift + Esc`：选择子元素
- `方向键`：切换同级元素
- `Shift + H`：显示/隐藏帮助面板
- `Double Esc`：退出
- `Double Cmd`：暂停/恢复选择器

## 安装

1. 打开 `chrome://extensions`
2. 开启 `Developer mode`
3. 点击 `Load unpacked`
4. 选择本项目目录
5. 点击扩展图标启用

## 提示

- 浏览器内部页面（如 `chrome://`）不允许注入脚本。
- 请在遵守站点条款与版权的前提下使用。
